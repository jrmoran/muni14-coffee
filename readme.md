# Muni14 Coffee

El Salvador Municipalities

# Usage

Donwload the file './build/muni14.js' and include it into your 
HTML file.

## Making queries 

    Muni14.get('06');     // "San Salvador"
    Muni14.get('0618');   // "Soyapango"

# Dev

Run `cake` to list available tasks

## Dependencies

* CoffeeScript
* jasmine-node
* js-yaml
